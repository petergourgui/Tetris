Tetris

This is an implementation of the game of Tetris. It is the classic falling block puzzle game implemented in Python using Pygame.
Players control falling jewels, trying to match three or more of the same type horizontally, vertically, or diagonally to clear
them from the game board. The game continues until the jewels stack up to the top of the board.

How to Run:

Make sure to install pygame using: pip install pygame.
Then run main.py and the game will begin.

Features:

Jewels represented by different colors
Control the falling jewels using keyboard inputs:
    Left arrow: Move the jewel left.
    Right arrow: Move the jewel right.
    Spacebar: Rotate the jewel.
Game Over screen displayed when the jewels reach the top
Resizable game window
